# HTML & CSS Course Projects

This repository contains code and projects created while following the **HTML & CSS Full Course - Beginner to Pro** online course.

## Overview

These files represent a personal learning journey through web development fundamentals, including:
* **YouTube Clone**: A responsive replica of the YouTube homepage (`youtube.html`) utilizing Grid, Flexbox, and complex CSS positioning.
* **CSS Layout Practice**: Exercises focused on specific layout techniques like CSS Grid (`grid.html`), Flexbox (`flexbox.html`), and Positioning (`position.html`).
* **Introductory Exercises**: Basic HTML/CSS concepts located in the `intro-to-html` folder.
