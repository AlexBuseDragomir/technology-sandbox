# Playground & Learning Area

Repository containing learning exercises, simple algorithmic problems, and smaller projects created while studying for interviews or learning new technologies.

## Project Modules

### Employee Web App (`/employee-web-app`)
A **Java Spring Boot** application for managing employee records.
* **Tech Stack:** Java, Spring Boot, Maven.
* **Focus:** Building RESTful APIs and handling database operations.

### HTML & CSS Course (`/html-css-course`)
A YouTube clone built to practice responsive design.
* **Source:** Code follows the instructions provided by **SuperSimpleDev** (YouTube).
* **Purpose:** 1:1 educational implementation to master CSS positioning, Flexbox, and Grid.

### Interview Problems (`/interview-problems`)
A collection of algorithmic challenges and solutions.
* **Languages:** Java & SQL.
* **Topics:** Multithreading, custom sorting, list processing, SQL queries etc.

### Bash Scripting (`/bash-scripting`)
Shell scripts and exercises exploring command-line automation and scripting basics.
